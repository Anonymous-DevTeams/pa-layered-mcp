    Search Guide#

    Advanced search syntax for PA-Retriever.

    Search Command#

    1
2
3
4
5
6

    pa search <query> [options]

Options:
  --limit=N     Limit results (default: 10)
  --json        JSON output
  --content     Include body content

    Basic Search#

    Keyword Search#

    1
2
3
4
5

    # Single keyword
pa search "login"

# Multiple keywords (implicit AND)
pa search "login auth"

    Field Search#

    Syntax#

    field:value

    Supported Fields#

    Field Description Example
    type Document type type:l2
    domain Domain domain:auth
    status Status status:active
    priority Priority priority:high
    author Author author:john
    tag Tag tag:jwt

    Examples#

    1
2
3
4
5
6
7
8

    # L2 type only
pa search "type:l2"

# auth domain only
pa search "domain:auth"

# Specific tag
pa search "tag:jwt"

    Complex Queries#

    AND Operation#

    1
2
3
4
5

    # Explicit AND
pa search "type:l2 AND domain:auth"

# Implicit AND (space)
pa search "type:l2 domain:auth"

    OR Operation#

    1
2
3
4
5

    # L1 or L2 type
pa search "type:l1 OR type:l2"

# auth or users domain
pa search "domain:auth OR domain:users"

    NOT Operation#

    1
2
3
4
5

    # Exclude deprecated
pa search "type:l2 NOT status:deprecated"

# Exclude drafts
pa search "domain:auth NOT status:draft"

    Parentheses Grouping#

    1
2
3
4
5

    # (auth OR users) domain L2s
pa search "type:l2 AND (domain:auth OR domain:users)"

# Complex combination
pa search "(type:l1 OR type:l2) AND domain:orders NOT status:draft"

    Range Search#

    Token Count#

    1
2
3
4
5

    # More than 1000 tokens
pa search "tokens:>1000"

# Less than 500 tokens
pa search "tokens:<500"

    Full-text Search (FTS)#

    Content Search#

    1
2
3
4
5

    # Body contains "suspend fun"
pa search "suspend fun"

# Body contains "interface"
pa search "interface"

    JSON Output#

    1
2
3
4
5
6
7
8

    # JSON format
pa search "type:l2" --json

# With jq
pa search "type:l2" --json | jq '.[] | .id'

# Save to file
pa search "domain:auth" --json > auth-docs.json
