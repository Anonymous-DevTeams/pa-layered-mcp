    Document-Code Connection#

    Automatic Connection#

    PA-Retriever attempts automatic connection using:

    1. Name matching: Document ID and signature name
    2. Marker matching: @pa-port marker

    Using Markers#

    Add marker to document for explicit connection:

    1
2
3
4
5
6
7
8
9

    <!-- L2-LoginPort.md -->
---
type: l2
domain: auth
---

# L2-LoginPort

<!-- @pa-port: LoginPort -->

    Add comment marker to code:

    1

    // LoginPort.kt

// @pa-doc: auth.L2-LoginPort
interface LoginPort {
    suspend fun login(request: LoginRequest): LoginResult
}
