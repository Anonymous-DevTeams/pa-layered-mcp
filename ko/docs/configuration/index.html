    설정하기#

    Claude, Obsidian, PA-Retriever 간의 연동 설정을 구성합니다.

    개요#

    ┌─────────────┐     MCP      ┌──────────────┐
│   Claude    │◄────────────►│ PA-Retriever │
│   Desktop   │              │  MCP Server  │
└─────────────┘              └──────┬───────┘
                                    │
                                    ▼
┌─────────────┐              ┌──────────────┐
│  Obsidian   │─────────────►│  mcp-docs/   │
│   (편집기)   │    파일      │  (Vault)     │
└─────────────┘              └──────────────┘

    연동 구성#

    도구 역할 연결
    Claude Desktop AI 에이전트 MCP → PA-Retriever
    PA-Retriever 문서 인덱서/검색 파일 → mcp-docs
    Obsidian 마크다운 편집기 직접 편집 → mcp-docs
