## .pa-index.yaml

Vault 루트에 생성:

```yaml
# mcp-docs/.pa-index.yaml
name: my-project

# 추가 타입 (선택)
types: []

# 제외 패턴
exclude:
  - "daily/*"
  - "attachments/*"
  - "*.excalidraw.md"

# 타입 추론 규칙 (선택)
infer:
  - pattern: "ports/**/L1-*.md"
    type: l1
  - pattern: "ports/**/L2-*.md"
    type: l2
  - pattern: "ports/**/API-*.md"
    type: api

    Dataview 쿼리 예시#

    모든 L2 Port 목록#

    TABLE domain, status, updated
FROM ""
WHERE type = "l2"
SORT domain, file.name

    도메인별 문서 수#

    TABLE length(rows) as "문서 수"
FROM ""
WHERE type
GROUP BY domain

    최근 수정된 문서#

    TABLE type, domain, updated
FROM ""
WHERE type
SORT updated DESC
LIMIT 10
