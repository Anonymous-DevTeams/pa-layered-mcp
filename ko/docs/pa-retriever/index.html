    PA-Retriever#

    PA-Layer 문서를 인덱싱하고 검색하는 CLI 도구입니다.

    개요#

    PA-Retriever는 다음 기능을 제공합니다:

    • 문서 인덱싱: 마크다운 문서를 파싱하고 SQLite에 저장
    • 고급 검색: AND/OR/NOT 쿼리 지원
    • MCP 서버: Claude와의 실시간 연동
    • 코드 인덱싱: 코드 시그니처 추출 및 문서 연결
    • Bundle 공유: 문서 패키징 및 배포
